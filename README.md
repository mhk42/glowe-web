# Glowe Skincare E-commerce Site

This is a fully responsive, single-page e-commerce storefront for a beauty
brand, **Glowe Skincare**, inspired by Korean Hanbang traditions. The site
showcases the brand's aesthetic and products in a clean, modern layout.
